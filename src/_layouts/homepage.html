The Mapbox Vision SDK
提前驶向未来。

未来出行，我们需要适合于机器和人的地图。动态地图、智能导航、增强现实、高性能计算机视觉技术将会变革人机与道路交互的方式。Mapbox Vision SDK 集革命性技术于一体，实时处理环境场景，把普通的车载摄像头变成洞察万物的“眼睛“，目之所及即是交互。

注：Mapbox Vision SDK 可与 Mapbox 优秀的实时交通与导航功能无缝衔接，Android 和 iOS 开发者能够在自己的原生 App 中轻松加入车辆抬头显示体验。在这里了解详细文档: iOS | Android

Mapbox Vision SDK 解锁了一套具有未来感的功能 —— 包括增强现实导航、物体检测、语义分割。在它的帮助下，普通的智能手机也能化身功能强大的车载智能传感器。

Mapbox Vison SDK将提供前所未有的实时道路数据，并接入Mapbox的实时定位平台。图像可以直接在用户设备上处理，因此当前的道路状况和实时事件数据能够低延迟，低带宽的环境下进行实时更新。在设备上运行高效的人工智能算法 - 无需再上传到云服务器中进行二次处理。Mapbox Vision SDK功能强大，能够反馈实时驾驶环境，同时兼容性强大，可在数十亿不同移动设备上运行。
